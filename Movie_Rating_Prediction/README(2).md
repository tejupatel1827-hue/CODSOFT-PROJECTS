# CodSoft Data Science Internship - Task 2: Movie Rating Prediction Using Python

This repository contains a complete Machine Learning project to predict **movie ratings** using features such as **Year, Duration, Votes, Genre, Director, and Actors**.

---

## 📂 Project Repository Structure

```text
Movie_Rating_Prediction/
│
├── Movie_Rating_Prediction.ipynb
├── IMDb Movie Ratings.csv
├── requirements.txt
└── README.md
```

---

## 🛠️ Technologies Used

- Python
- Google Colab
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

---

## 🤖 Machine Learning Algorithm

**Random Forest Regressor**

---

## 📊 Project Workflow

1. Import Libraries
2. Load Dataset
3. Data Cleaning & Preprocessing
4. Exploratory Data Analysis (EDA)
5. Data Visualization
6. Feature Engineering
7. Label Encoding
8. Train-Test Split
9. Model Training
10. Prediction
11. Model Evaluation

---

## 📈 Evaluation Metrics

- R² Score
- Mean Absolute Error (MAE)
- Root Mean Squared Error (RMSE)

---

## ▶️ Installation & Execution

```bash
pip install -r requirements.txt
```

Open **Movie_Rating_Prediction.ipynb** in Google Colab or Jupyter Notebook and run all cells.

---

## 💡 Project Outcome

The Random Forest Regressor model predicts movie ratings using movie features and demonstrates the complete Machine Learning workflow from preprocessing to evaluation.
